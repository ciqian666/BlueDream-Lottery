# 💎 BlueDream Lottery v1.0.6 更新日志

## 🚀 核心更新内容

### 1. 🌍 全版本极致兼容性
*   **兼容层架构**：（1.7.10 - 1.21.x）的极致兼容

### 2. 📊 工业级统计与排行榜系统 (Statistics 2.0)
*   **多引擎数据库支持**：集成 HikariCP 连接池，支持 **SQLite, MySQL, MariaDB, PostgreSQL**
*   **异步数据引擎**：所有统计计算与日志记录均在独立线程完成，TPS 零损耗
*   **数据库开关控制**：新增了数据库启用/禁用切换功能，默认不启用以保证开箱即用
*   **7 大可视化 GUI 界面**：
    *   **全局概览**：实时监控全服抽奖流水、金币/点券流向
    *   **四大排行榜**：抽奖榜、消耗榜、大奖榜，以及独创的 **"欧皇榜"**（基于真实爆率算法）
    *   **热门分析**：统计最受欢迎的奖池与掉落频率最高的物品
    *   **系统监控**：实时查看数据库连接状态与配置信息

### 3. 🎨 视觉与交互体验升级
*   **大奖公告优化**：抽中“终极大奖”时的全服广播现在支持自定义显示名称（默认统一显示为“超级大奖”），避免显示原始 ID
*   **粒子效果适配**：重构了粒子系统，确保在 1.7.10 也能看到绚丽的粒子螺旋特效
*   **动画引擎重构**：现在能够精准记录每次抽奖的消耗与产出，为统计系统提供底层支持

### 4. 🛠️ 指令系统与稳定性增强
*   **热重载支持**：`/lt reload` 现在可以智能重载数据库连接，修改配置后无需重启
*   **错误日志清理**：优化了 SQLite 驱动的探测逻辑
*   **新指令**：`/lt stats` (或 `/lt ranking`) —— 开启可视化统计中心
*   **权限细化**：新增了对应的权限节点，方便服主进行精细化管理
