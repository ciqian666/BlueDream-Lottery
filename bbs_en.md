BlueDream-Lottery

- 🎬 Visual Feast: 14 built-in animation styles (e.g., Slot Machine, Dimension Rift, Thunder Strike, Time Travel), independently configurable for each pool.
- 💎 Lottery Modes: Supports Left-click for single roll and Right-click for 10-rolls, with automatic cost calculation and smooth transitions.
- ⚖️ Pity System: Supports "Grand Prize" settings and pity counters. Ensure rewards for players even during bad luck streaks.
- 🔗 Command Rewards: Items can trigger multiple console commands (e.g., granting money, points, or permissions) in addition to physical items.
- 💰 Multiple Payments: Fully compatible with Vault (Economy), PlayerPoints, and custom Key Items.
- 🌍 Physical Entrances: Supports Physical Lottery Blocks! Set any block in the world as an entrance with full protection against griefing and explosions.
- 🛠️ Full GUI Editor: No more manual config editing! Manage pools, adjust probabilities, and switch animations directly in-game via a user-friendly interface.
- 🚀 Extreme Performance: Utilizes independent player data files and asynchronous snapshot saving to ensure zero main-thread lag.
- 🛡️ Smart Protection: Full inventory? No problem!  Rewards are automatically dropped at the player's feet with a helpful notification.
- 📦 Portable Chests: Administrators can issue specific "Lottery Chest" items that players can right-click to start drawing anywhere.
- 🎵 Immersive Audio: All animations feature sound feedback to make every roll feel like a celebration.
- 🌟 Cool Particles: 5 stunning ambient particle effects (Vortex, Halo, Star Dust, etc.), switchable via Admin GUI with auto-save.
- 🌍 Internationalization: Dynamic language switching between zh_cn and en_us with fully customizable messages.

📝 Commands & Permissions
- /lt help - View full help menu
- /lt create <name> - Create a new pool (Permission: bluedream.lottery.admin)
- /lt remove <name> - Delete a specific pool (Permission: bluedream.lottery.admin)
- /lt edit <name> - Open the GUI editor (Permission: bluedream.lottery.admin)
- /lt setblock <name> - Set the target block as a lottery entrance (Permission: bluedream.lottery.admin)
- /lt removeblock - Remove the lottery setting from the target block (Permission: bluedream.lottery.admin)
- /lt play <name> - Open the lottery GUI and start drawing (Permission: bluedream.lottery.use) (Default for players)
- /lt reload - Reload configuration files (Permission: bluedream.lottery.admin)
- /lt lang <zh_cn|en_us> - Switch plugin language (Permission: bluedream.lottery.admin)
