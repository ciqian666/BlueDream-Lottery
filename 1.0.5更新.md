💎 BlueDream Lottery v1.0.5 更新公告

✨ 核心更新：全版本跨代兼容性支持

- **新增：全版本 API 适配层 (Adapter)**
  - 完美支持从 **Minecraft 1.7.10** 到 **1.21.x** 的所有主流版本。
  - 自动适配 1.13+ 的材质扁平化（Flattening），解决旧版 ID 与新版材质名的冲突。
  - 适配 1.9+ 的双手（Offhand）交互机制。

- **新增：粒子效果跨版本支持**
  - 重写了粒子渲染引擎，在 1.9+ 使用原生 Particle API，在 1.8 及以下版本自动回退至 Effect API。
  - 解决了旧版本缺少某些粒子类而导致的崩溃问题。

- **优化：Java 版本兼容性**
  - 编译目标降至 **Java 1.8**，确保在旧版 Java 环境（如 1.7.10 常用环境）中也能稳定运行。
  - 修复了所有 Java 9+ 的新语法特性（如 `List.of`, `Map.of`, `Files.writeString` 等），改用 Java 8 兼容实现。

- **修复：1.9+ 爆炸事件兼容**
  - 针对 `BlockExplodeEvent` 进行了动态类检测与异步注册，防止在不含此事件的旧版本中加载失败。

- **优化：代码健壮性**
  - 完善了 GUI 中的空指针检查，适配了不同版本 `ItemMeta` 的差异。
  - 移除了所有硬编码的音效名，统一使用兼容性映射表。
